getIssuesForBuild

fun getIssuesForBuild(plankey: String, buildnumber: String): ArrayList<String>

Funktion zum Auslesen der beinhalteten Jira-Issues in einem Bamboo-Build

  • Auslesen der Bamboo-Build-Infos , expandiert um die Jira-Issues
  • Auswertung des JSON-Objekts
  • Rueckgabe der beinhalteten JIRA-Issues in einer Liste

Parameters

plankey - Der Bamboo-PlanKey (z.B. "MCBS-MR63") zu dem die Issues ausgelesen werden sollen

buildnumber - Die laufende Nummer des Bamboo-Builds zu dem die Issues ausgelesen werden sollen

Return
issueList Eine Liste von Issues, die laut Bamboo in diesem Build neu hinzugekommen sind
