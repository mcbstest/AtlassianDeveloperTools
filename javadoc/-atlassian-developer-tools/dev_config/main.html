fun main(args: Array<String>): Unit

Methode zur Abfrage , ob ein Issue Config-Abhängigkeiten unterliegt

  • die Komponenten des Issue werden ausgelesen
  • Enthält die Komponentenliste "ms-configuration" wird "TRUE" im Config-Propertyfile gesetzt, ansonsten false

Aufruf : java -cp ./build/libs/AtlassianDeveloperTools-all-1.0.jar:. dev_config.CheckConfigKt https://jira.freenet-group.de/rest/api/2/ mcbstest:password MCBS-2774

Parameters

bambooURL - die URL fuer produktiven Bamboo-Zugriff

jiraURL - die URL fuer produktiven JIRA-Zugriff

credentials - login:password fuer produktiven JIRA-Zugriff

issue - das zu prüfende Ticket

exceptionFile - Property-File mit nicht zu beruecksichtigenden Prefixes (UTF , M;SBS etc.) (erwartet in jiraResources)
