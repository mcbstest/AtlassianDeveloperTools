addMilestone

fun addMilestone(prjid: String, number: String, name: String, dev: String, git: String, ibn: String): Unit

Funktion zum Anlegen eines TestRail-Milestones

  • Name-Pattern : "{Sprintnumber}-IBN! {name}
  • Description : Dev : xx-yy , GIT: xx-yy
  • Due_on : IBN

Parameters

prjid - Die Id des Projektes , in dem der Milestone angelegt wird

number - Die MCBS-Core-Sprintnummer (wie in Jira)

name - Der Name des Sprints im Klartext

dev - Der Entwicklungszeitraum als String

git - Der Testzeitraum als String

ibn - Der Inbetriebnahmetermin
