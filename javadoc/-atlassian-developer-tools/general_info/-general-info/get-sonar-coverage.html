getSonarCoverage

fun getSonarCoverage(sonarURL: String, sonarCompA: String, sonarCompP: String, metricKey: String): String

Methode zur Ermittlung der TestCoverage einer Komponente via Sonar

Parameters

sonarURL - : Die Sonar-Basis-URL

sonarCompA - : Die Applikation (so wie in Sonar hinterlegt)

sonarCompP - : Der "Pfad" zur Applikation (so wie in Sonar hinterlegt)

metricKey - : Die abzufragende Metric (i.a. coverage)

Author
Bernd Moeller

Return
Eine Zeichenkette, die den prozentualen Wert ohne Zusätze enthält
