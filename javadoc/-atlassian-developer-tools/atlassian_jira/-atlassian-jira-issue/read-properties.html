readProperties

fun readProperties(propFile: String): Properties

Einlesen eines Property-Files aus dem jiraResources-Verzeichnis. Das Property-File enthält unter dem key "components" eine Liste der zum Projekt gehörenden Komponenten Der Name ergibt sich aus einer übergebenen Variablen, die im Bamboo-Kontext als Variable an das Skript übergeben werden muss. Selbst erzeugte Konvention : "'Bamboo-Projekt'-'Bamboo-Plan'

Parameters

propFile - Der Name des Property-Files in der Konvention 'Bamboo-Projekt'-'Bamboo-Plan' ohne Pfad oder Extension
