getSemanticVersionForReleaseNew

fun getSemanticVersionForReleaseNew(version: String): String

Methode zur Ermittlung der SemanticVersion einer Jira-FixVersion

Es werden die Issues abgefragt, die über die übergebene FixVersion verfügen. Aus diesen Issues wird die höchste SemanticVersion aus den Keywords extrahiert.

Parameters

version - : Die Jira-FixVersion, deren SemanticVersion ausgelesen werden soll

Author
: Bernd Möller

Return
Eine Zeichenkette, die die SemanticVersion des Release enthält
