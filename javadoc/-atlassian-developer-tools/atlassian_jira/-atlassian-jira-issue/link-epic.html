linkEpic

fun linkEpic(issue: String, epic: String): Unit

Funktion zum Setzen der Epos-Version an einem Jira-Issue (customfield_13790). Sofern der Jira-Issue bereits geschlossen ist, wird die Aktion auf der Konsole mit "Fehler !" quittiert

Parameters

issue - Das Jira-Ticket

epic - Die uebergebene EPOS-Version, die verlinkt werden soll.
