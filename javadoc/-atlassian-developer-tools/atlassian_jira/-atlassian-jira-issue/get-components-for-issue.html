fun getComponentsForIssue(issue: String): ArrayList<String>

Funktion zum Auslesen der Komponenten eines Jira-Issues

  • Auswertung des JSON-Objects
  • Rueckgabe der einzelnen Komponenten in einer Liste

Parameters
