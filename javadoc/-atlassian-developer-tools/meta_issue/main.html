Methode um JIRA-META-Issues zu erzeugen und zu beobachten

  • Erzeugung des Issue im Projekt "META-MCBS"
  • Beobachter werden gesetzt Aufruf : java -cp ./devtools/apps/atltools/AtlassianDeveloperTools-all- ....

Parameters

jiraURL - die URL fuer produktiven JIRA-Zugriff

credentials - login:password fuer produktiven JIRA-Zugriff

filename - csv-File mit Sprint-Informationen

watchers - eine komma-separierte Liste der anzulegenden Beobachter
